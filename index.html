Trans Lingua Script

Developed a model which can detect the signboards, extract the text from the signboards and convert it into Hindi language.


Developed an object detection model to detect sign board objects in a given image and a CNN-RNN encoder decoder model to convert the cropped signboard into text and later used a sequence to sequence with attention model for transliterating the above sequence of characters from English to Hindi language.


Achieved an accuracy of around 90% and developed using Python, TensorFlow, Keras and Natural Language Processing.
